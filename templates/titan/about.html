About

This Project and website is the final project for IS 403. It matches Tutors with students and shows general information about tutors. You can also edit their details.

Features of this Website Include

  • Home Page
    • General Information
    • Contact Button for comments and further help
  • About
    • General Information about the devlopment team and this website
  • Search
    • Search By tutor name, state, credential, speciality, & gender (All Nullable) (Generates Interactable List)
    • Add a tutor (Takes to a new page)
  • Tutor Detail Page
    • View important information about a tutor
      • Name
      • State
      • Degree
      • Gender
      • Subjects
    • Create Tutor information
    • Delete Tutor information
    • View students they tutor
    • Navigate to students detail pages
